Plot the histogram corresponding to the provided key.

Parameters
[in]keyThe key of the histogram to plot.
[in]parThe corresponding pad that the histogram should be plotted on

The main plotting method. This method determines arguments to be passed to TTree::Draw() to handle plotting. If a histogram has previously been created the Draw command only adds new entries after that last draw to reduce time required to loop over the tree.
